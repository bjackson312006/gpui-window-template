This repo is a project template for a GPUI app that renders its own custom window decorations. It handles window controls/dragging/resizing correctly across Windows, MacOS, and Linux. It's also able to recognize if it's in tiled/multitasking mode (including on Wayland), which is used to determine if the window should render with rounded corners or not. Because the titlebar is fully customizable, it can be extended with extra custom buttons/menus/etc.
